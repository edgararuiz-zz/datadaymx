El paquete de R llamado sparklyr facilita el aprendizaje mediante un instalador de Spark que se puede usar dentro de una computadora personal, incluyendo Windows. En este taller los participantes aprendern a utilizar Spark por medio de R mediante el uso de diferentes tcnicas y funciones para:

  • Transformar datos
  • Crear modelos estadsticos
  • Programar canales de datos.

libro <- spark_read_text(sc, "libro", "sense.txt")
libro <- libro %>%
  mutate(
    line = regexp_replace(line, "[_\"\'():;,.!?\\-]", " ")
    ) 
palabras <- libro %>%
  ft_tokenizer(input.col = "line",
               output.col = "lista")
palabras <- palabras %>%
  ft_stop_words_remover(input.col = "lista",
                        output.col = "lista_limpia")
